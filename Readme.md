## 기상 인증
오전 07시 마다 `기상 인증 배열` 초기화

특정 커맨드와 함께 글을 올리면 해당하는 유저 이름 구글 시트에 O 표시와 함께 `기상 인증 배열`에 담기

오전 09시 10분에 배열에 없는 사람들 알림 메세지 보내주기

오전 09시 20분에 배열에 없는 사람들은 X로 체크
